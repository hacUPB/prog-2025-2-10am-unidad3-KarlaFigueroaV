# Tarea
## Tabla comandos aritmeticos
| Operador | Explicación                          |
| -------- | ------------------------------------ |
| +    | Suma                                 |
| -      | Resta                                |
| *      | Multiplicación                       |
| /      | División decimal                     |
| //     | División entera (descarta decimales) |
| %      | Módulo (residuo)                     |
| **     | Potencia                             |
 
 ## Tabla comandos relacionales
| Operador | Explicación   |
| -------- | ------------- |
| ==     | Igual a       |
| !=     | Diferente de  |
| >      | Mayor que     |
| <      | Menor que     |
| >=     | Mayor o igual |
| <=     | Menor o igual |
